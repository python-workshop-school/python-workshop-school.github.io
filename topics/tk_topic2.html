Віджет Label

до цеху tkinter

Теоретичний матеріал

Віджет Label – клас мітки. Відображає текст у вікні і служить в основному для інформаційних цілей (вивід повідомлень, підпис інших елементів інтерфейсу тощо).

Загальний синтаксис:

name = Label(windows)

де, name – ім'я мітки, а window – ім'я вікна, на якому вона розташовується

Властивості Label::

Властивість Значення
width Ширина мітки
height Висота мітки
text Текст на мітці
bg Фон мітки
fg Колір тексту
bd Ширина межі мітки
activebackground Колір фону (коли мітка натиснута)
activeforeground Колір тексту (коли мітка натиснута)
disabledbackground Колір фону (коли властивість state == DISABLED)
disabledforeground Колір тексту (коли властивість state == DISABLED)
state Стан мітки (NORMAL, DISABLED)
compound Розташування картинки на мітці (CENTER, BOTTOM, LEFT, RIGHT, TOP)
justify Вирівнювання тексту
relief Рельєф мітки (FLAT, GROOVE, RIDGE, SUNKEN, RAISE)
overrelief Рельєф мітки, коли над нею знаходиться курсор (FLAT, GROOVE, RIDGE, SUNKEN, RAISE)
image Ім'я картинки, яку буде відображено на мітці
font Вид шрифту на мітці
textvariable Ім'я змінної, в якій буде зберігатися текст на мітці

Змінити властивість мітки або будь-якого іншого віджета можна так:

viget_name["назва_властивості"] = "значення"
viget_name.config(назва_властивості = "значення")

Приклад

Створимо у вікні за допомогою двох міток зображення прапора України з намисом «Слава УКРАЇНІ»

Код програми 

from tkinter import *
win=Tk()
win.title('Прапор України')
win.geometry('641x306')
win.resizable(False, False)
lab_1=Label(win, font='Times 48', width='18', height='2', bg='blue2', fg='gold', text='Слава').pack()
lab_2=Label(win, font='Times 48',width='18', height='2', bg='gold', fg='blue2', text='УКРАЇНІ').pack()
win.mainloop()

Результат

Завдання

  1. Напишіть код програми, в якому створіть мітки за зразком.

  2. Файл з кодом програми збережіть у власну папку з іменем tk_2_01.py;
  3. Назви кольорів для для міток: red, orange, yellow, green, cyan, bleu, purple;
  4. Назви кольорів для для напису міток: pink, indianred4, yellow4, green2, steelblue, steelblue1, mediumpurple1;

* для горизонтального відображення віджетів у вікні програми ми використали метод .pack(side=LEFT, fill=Y) з його властивостями, про них більш детально дізнаємося в наступній темі.
